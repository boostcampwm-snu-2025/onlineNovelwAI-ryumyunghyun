## 서비스 목적
- 많은 사람들이 웹소설을 작성해서 올리지만 처음에은 독자를 모으기 어렵고 피드백을 받기 어렵습니다. 그래서 초기에 AI를 통한 가상 독자를 만들어 나아갈 방향이나 피드백을 받으면 좋을 것 같습니다.
- 또한 가상으로 투자하거나 영화를 만드는 게임 같이 시뮬레이션 게임들이 있는데 여기서 조금 더 나아가 직접 소설을 쓰고 평가받는 게임을 만들어 보고 싶습니다
## 서비스 타겟
- 처음 소설을 쓰기 시작하는 사람
- 조금더 현실감 있는 시뮬레이션 게임을 원하는 사람

## 핵심가치
- 유저가 쓴 글을 AI가 각각의 페르소나를 가지고 평가/리뷰 남긴다
- 올라온 소설 내용을 기억하고 각각의 AI가 평가한다
- 변화하는 지표를 유저가 확인할 수 있다
### Painpoint
- 직접 소설을 써야한다
### Wowpoint
- 우상향할 경우 재미가 있다
